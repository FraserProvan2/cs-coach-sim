# CS Ultimate Coach

An online Counter-Strike team builder web app, with Player vs Player turn based combat.

Built using Laravel 7, Vue.js, Pusher, Bootstrap (+Bootswatch). All player data is owned by
[HLTV](https://www.hltv.org/).
